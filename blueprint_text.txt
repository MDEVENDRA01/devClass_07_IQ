Especially valuable for first-time users.Uses a lightweight extraction prompt (separate from the main evaluation prompt).UI: 'Quick Input' toggle above the form — switches between full form and single-text-field modeBackend: Extraction prompt parses free text → returns structured JSON → auto-populates React stateUser can review and edit auto-filled values before generating11.2 Lesson Plan Version ComparisonTeachers can regenerate feedback after improving their lesson plan, then compare side-by-sideUI: 'Compare' button in History — opens 2-column view: Original vs Revised with diff highlightsQuality score delta shown prominently: e.g. '↑ +2.1 quality improvement'11.3 Department BenchmarkingHOD can view anonymised quality score distribution across all teachers in their departmentBell curve chart showing individual score vs department average vs institution averagePrivacy: individual teacher names hidden from peers;
